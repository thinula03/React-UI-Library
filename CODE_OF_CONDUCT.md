# Code of Conduct

This project is intended to be a welcoming, useful place for people building UI.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent.
- Give actionable feedback.
- Keep discussions focused on the project.
- Make space for beginners and experienced contributors alike.

## Unacceptable Behavior

- Harassment, insults, or personal attacks
- Discriminatory language
- Trolling or repeated bad-faith arguments
- Publishing private information
- Dismissing people instead of discussing ideas

## Enforcement

Maintainers may remove comments, close issues, or block participants who make
the project unsafe or unproductive.

If you see a problem, contact the maintainers privately.
